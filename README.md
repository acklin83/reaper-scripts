# Frank Acklin's REAPER Scripts

ReaPack repository for REAPER scripts by Frank Acklin.

## Installation

1. In REAPER: **Extensions → ReaPack → Import repositories**
2. Paste this URL:
   ```
   https://github.com/acklin83/reaper-scripts/raw/main/index.xml
   ```
3. **Extensions → ReaPack → Browse packages** to install scripts

## Scripts

### RAPID
Recording Auto-Placement & Intelligent Dynamics — automates track mapping, media import, and LUFS normalization.
